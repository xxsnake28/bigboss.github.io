#8 Amazon面试题

发表于 2019-12-18 更新于 2019-12-19 分类于 interview-question 阅读次数： 评论次数：

题目

从A点到目标B点，中间有N个节点，每次可以移动x或者y或者z个节点。

要求：
写一个函数，输出所有可能的不重复的从A到B的路径。

例子

例1

假设节点数N = 4，可移动的方式为 S = [1, 2]

那么输出的结果如下：

  • 1，1，1，1
  • 2，1，1
  • 1，2，1
  • 1，1，2
  • 2，2

例2

假设节点数 N = 5，可移动的方式为 S = [1, 3, 5]

那么输出的结果如下：

  • 1，1，1，1，1
  • 1，1，3
  • 1，3，1
  • 3，1，1
  • 5

正则验证邮箱，手机号码，身份证号等信息

发表于 2019-12-17 分类于 note 阅读次数： 评论次数：
手机号

^1[3456789]\d{9}$ 或者 ^1(3|4|5|6|7|8|9)\d{9}$

固定号码

^(\(\d{3,4}\)|\d{3,4}-|\s)?\d{7,14}$

身份证号

15位：^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$
18位：^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{4}$
合并： (^\d{15}$)|(^\d{17}([0-9]|X)$)

匹配URL

(h|H)(r|R)(e|E)(f|F) *= *('|")?(\w|\\|\/|\.)+('|"| *|>)?

邮件地址

\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*

图片链接

(s|S)(r|R)(c|C) *= *('|")?(\w|\\|\/|\.)+('|"| *|>)?

IP地址

(\d+)\.(\d+)\.(\d+)\.(\d+)

中国邮政编码

[1-9]{1}(\d+){5}

整数

\d+

浮点数（即小数）

(-?\d*)\.?\d+

任何数字

(-?\d*)(\.\d+)?

中文字符串

[\u4e00-\u9fa5]*

双字节字符串 (汉字)

[^\x00-\xff]*

要求：
写一个函数，从集合set中找到所有以s开头的字符串。
